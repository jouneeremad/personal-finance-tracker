# Personal Finance Tracker

A Python application that helps users track income, expenses, and personal finances.

## Features

- Add income
- Add expenses
- View transactions
- Calculate balance
- Expense categories
- Save financial data automatically

## Technologies Used

- Python
- File Handling

## Future Improvements

- Monthly reports
- Charts and graphs
- Budget planning
- Expense analytics
- Export reports
